# 脑出血智能诊疗数据集建设：会议投屏版

## 会议希望交流的重点

这次交流，主要希望向临床老师请教并对齐以下 4 件事：

1. 第一阶段到底做哪类脑出血。
2. 脑出血从发病到复诊的真实临床闭环是什么。
3. 在这个闭环中，哪些环节最值得优先做 AI 辅助。
4. 第一阶段数据集最少要收哪些数据、哪些标签，以及医院侧适合提供什么协作支持。
