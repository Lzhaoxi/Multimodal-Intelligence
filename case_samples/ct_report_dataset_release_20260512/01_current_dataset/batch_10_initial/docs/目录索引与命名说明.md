# AI诊疗课题目录索引与命名说明

## 目录结构

当前病例样例按治疗方式分为两类：

- `手术治疗`
- `保守治疗`

每个病例目录一般以：

`姓名 + 病案号 + 疾病类型`

的形式命名。

## 结构特征

### 手术治疗

常见文件：

- `入院CT图` 或 `入院CT图像`
- `入院CT报告`
- `复查CT图` 或 `复查CT图像`
- `复查CT报告` 或 `出院CT报告`
- `首次病程记录`
- `出院记录`
- `手术记录`
- `手术同意书`（部分病例）

### 保守治疗

常见文件：

- `入院CT图` 或 `入院CT图像`
- `入院CT报告`
- `复查CT图` 或 `复查CT图像`
- `复查CT报告`、`CT复查报告` 或 `7dCT报告`
- `首次病程记录`
- `出院记录`
- `谈话记录`

## 最稳定区分规则

如果只看文件结构，当前最稳定的区分特征是：

- 出现 `手术记录`，可优先视为 `手术治疗`
- 出现 `手术同意书`，可进一步支持 `手术治疗`
- 出现 `谈话记录` 且没有手术相关文件，优先视为 `保守治疗`

## 命名不一致清单

以下名称在语义上接近，但当前写法不统一：

- `入院CT图` / `入院CT图像`
- `复查CT图` / `复查CT图像`
- `7dCT图`
- `CT复查报告`
- `首次病程记录1` / `首次病程记录2`
- `出院记录1` / `出院记录2`

## 配套索引

完整文件索引见：

- `病例文件索引.csv`

其中列出了每例病例的：

- 治疗方式
- 病例目录名
- 文件总数
- 是否包含手术记录
- 是否包含手术同意书
- 是否包含谈话记录
- 是否包含入院 CT 视频
- 是否包含复查 CT 视频
- 是否包含入院 CT 报告
- 是否包含复查 CT 报告
- 文件名汇总
